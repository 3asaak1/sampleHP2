About

ワークショップって、どんな体験？

🌸

inimu のワークショップでは、好きな香りを試しながら、 自分だけのオリジナル香りアイテムづくりを楽しめます。 香りの知識がなくても、スタッフがやさしく案内するので大丈夫です。

「なんとなく好き」から選び始めても、少しずつ自分の好みが見えてきます。 完成したアイテムは、浅草での思い出として持ち帰れるのも魅力です。

Flow

当日の流れ

来店から完成までの流れを、4ステップで分かりやすく紹介します。

01
📝

受付・ご案内

予約時間に来店し、受付を行います。 スタッフからその日の流れや体験のポイントを説明します。

02
👃

香りを試して選ぶ

いくつかの香りを試しながら、自分に合いそうな香りを選びます。 迷っても相談しながら進められます。

03
🧪

ブレンドして調整

選んだ香りを組み合わせて、バランスを見ながら調整します。 少しずつ試すことで、完成イメージがつかみやすくなります。

04
🎁

完成・持ち帰り

自分だけの香りアイテムが完成。 その日の思い出として持ち帰り、自宅でも楽しめます。

Fragrance Basics

香りづくりの基本

香りは大きく3つの層で感じ方が変わります。まずはこの考え方だけ知っておけば安心です。

Top Note 最初に感じる香り
Middle Note 香りの中心になる印象
Base Note 最後まで残る落ち着いた香り
🍋

Top Note

最初の印象を決める香り。明るさや軽さを感じやすい部分です。

🌼

Middle Note

香り全体の主役になる部分。自分らしさを感じやすい中心の香りです。

🌳

Base Note

余韻として長く残る香り。落ち着きや深みを出したいときに大切です。

安心ポイント

初めてでも心配しなくて大丈夫

🙋

スタッフがサポート

迷ったときは相談しながら選べるので、知識がなくても進めやすいです。

⏱

長すぎない体験時間

45〜60分程度を目安に考えやすく、観光の合間にも組み込みやすい構成です。

👜

準備の負担が少ない

特別な道具を自分でそろえなくても参加しやすい体験として案内できます。

Concerns

よくある不安

Q

香りのことを全然知らなくても大丈夫？

大丈夫です。難しい知識よりも「好き」「心地いい」を大切にしながら選べます。

Q

一人で参加しても浮きませんか？

一人で体験を楽しむ人も多く、落ち着いて自分のペースで進めやすい内容です。

Q

時間内にうまく決められるか心配です

段階的に試しながら進めるので、いきなり正解を決めなくても大丈夫です。

Checklist

  • 予約日時を確認する

  • 気になることがあればFAQも読む
